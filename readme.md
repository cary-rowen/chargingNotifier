# Charging Notifier

An add-on for NVDA that allows you to customize AC power status alerts using voice announcements, sound notifications, or both.

## Features

* **Sound Notifications**: Plays distinct sound effects when your device is connected to or disconnected from AC power.
* **Configurable**: Choose between voice announcements, sound notifications, or both to suit your preference.

## Settings

The add-on adds a **Charging Notifier** category to NVDA's Settings dialog (`NVDA+N` → Preferences → Settings).

### Charging status notification type

* **Voice**: Uses NVDA's standard voice announcements for power status changes.
* **Sound** (default): Plays audio cues instead of voice announcements.
* **Voice and Sound**: Provides both audio cues and voice announcements.
