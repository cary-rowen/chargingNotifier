# Charging Notifier (充电提醒)

本插件增强了 NVDA 的电源提醒功能，允许你根据个人喜好，将电源插拔提醒设置为语音播报、声音音效或两者兼有。

## 功能特性

* **音效提醒**：当电脑连接或断开电源时，播放独特的提示音。
* **灵活配置**：可在语音播报、声音音效之间自由切换，也支持同时开启。

## 设置选项

在 NVDA 设置对话框（`NVDA+N` -> 选项 -> 设置）中，可以找到 **充电提醒** 分类。

### 充电状态通知类型

* **语音**：使用 NVDA 原生的语音提示。
* **声音**（默认）：使用音频提示音代替语音播报。
* **语音和声音**：同时使用语音和音效提醒。
